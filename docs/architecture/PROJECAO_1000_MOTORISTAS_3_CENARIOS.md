# 📊 PROJEÇÃO FINANCEIRA - 1000 MOTORISTAS ONLINE DIARIAMENTE

**Data:** 29/01/2025  
**Base:** 1000 motoristas online diariamente  
**Objetivo:** Projetar quantidade de corridas e lucro operacional em 3 cenários

---

## 🎯 PARÂMETROS BASE

### **Distribuição de Corridas por Faixa:**
- **30%** das corridas < R$ 10,00 → Taxa: **R$ 0,79**
- **40%** das corridas entre R$ 10,00 - R$ 20,00 → Taxa: **R$ 0,99**
- **30%** das corridas > R$ 20,00 → Taxa: **R$ 1,49**

### **Custo Operacional por Corrida:**
- **R$ 0,127752** (custo fixo de infraestrutura)

---

## 📊 CENÁRIO PESSIMISTA

### **Premissas:**
- **Motoristas online:** 1.000 diários
- **Corridas por motorista/dia:** 2,5 corridas (baixa demanda, período inicial)
- **Dias úteis no mês:** 30 dias
- **Utilização:** Motoristas trabalhando apenas em horários de baixa demanda

### **Cálculo de Volume:**
```
Corridas por dia: 1.000 motoristas × 2,5 corridas = 2.500 corridas/dia
Corridas por mês: 2.500 × 30 dias = 75.000 corridas/mês
```

### **Distribuição das 75.000 corridas:**
```
Faixa 1 (< R$ 10,00): 30% × 75.000 = 22.500 corridas
Faixa 2 (R$ 10-20):   40% × 75.000 = 30.000 corridas
Faixa 3 (> R$ 20,00): 30% × 75.000 = 22.500 corridas
```

### **💰 CÁLCULO DA RECEITA:**
```
Faixa 1: 22.500 × R$ 0,79 = R$ 17.775,00
Faixa 2: 30.000 × R$ 0,99 = R$ 29.700,00
Faixa 3: 22.500 × R$ 1,49 = R$ 33.525,00
────────────────────────────────────────
RECEITA TOTAL: R$ 81.000,00
```

### **💸 CÁLCULO DO CUSTO:**
```
75.000 corridas × R$ 0,127752 = R$ 9.581,40
```

### **📈 LUCRO OPERACIONAL:**

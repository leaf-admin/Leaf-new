# Scripts do Leaf Dashboard

## 📋 Lista de Scripts Disponíveis

### 🚀 Scripts de Inicialização

#### `start-dashboard.bat` - INICIAR DASHBOARD
```batch
start-dashboard.bat
```
**Função:** Inicia o dashboard com verificação automática de dependências
- ✅ Verifica se as dependências estão instaladas
- ✅ Executa verificação do TypeScript
- ✅ Inicia o servidor de desenvolvimento
- ✅ Abre em http://localhost:3000

---

### 🔧 Scripts de Correção

#### `fix-dashboard.bat` - CORRIGIR PROBLEMAS
```batch
fix-dashboard.bat
```
**Função:** Corrige problemas comuns automaticamente
- 🧹 Limpa cache do npm
- 📦 Remove node_modules e package-lock.json
- 🔄 Reinstala todas as dependências
- ✅ Verifica TypeScript
- 💡 Fornece instruções para iniciar

---

### 🔍 Scripts de Diagnóstico

#### `check-status.bat` - VERIFICAR STATUS
```batch
check-status.bat
```
**Função:** Verifica o status completo do sistema
- 📊 Versão do Node.js e npm
- 📦 Status das dependências
- ✅ Verificação do TypeScript
- 🌐 Status da porta 3000
- 💡 Sugestões de próximos passos

#### `diagnose-dashboard.bat` - DIAGNÓSTICO COMPLETO
```batch
diagnose-dashboard.bat
```
**Função:** Diagnóstico detalhado do sistema
- 📋 Lista todas as dependências
- ✅ Verificação do TypeScript
- 🎨 Verificação do Tailwind CSS
- ⚙️ Verificação do React Scripts
- 📊 Relatório completo

---

## 🎯 Como Usar

### 1. Primeira Execução
```batch
check-status.bat
```
Verifica se tudo está configurado corretamente.

### 2. Se Houver Problemas
```batch
fix-dashboard.bat
```
Corrige problemas automaticamente.

### 3. Iniciar Dashboard
```batch
start-dashboard.bat
```
Inicia o dashboard com verificação automática.

### 4. Diagnóstico Detalhado
```batch
diagnose-dashboard.bat
```
Executa diagnóstico completo se necessário.

---

## ⚠️ Observações Importantes

- **Execute os scripts como administrador** se houver problemas de permissão
- **Feche o VS Code** antes de executar `fix-dashboard.bat`
- **Aguarde a conclusão** de cada script antes de executar o próximo
- **Verifique a porta 3000** se o dashboard não iniciar
